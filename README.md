## Kurzbeschreibung
Dieses Repository ist eine persönliche Sammlung meiner wichtigsten Projekte aus Studium und Praxis. Es umfasst:

- Unity-Entwicklung und 3D-Spiele/Simulationen  
- KI-gestützte NPC-Systeme (Verhalten, Entscheidungslogik)  
- 3D-Modelle & Assets (Blender, PBR, Motion Capture)  
- Automatisierungsskripte für Workflows und Datenverarbeitung  
- Tools zur Analyse, Optimierung und Dokumentation von Prozessen  

Ziel ist es, Lösungen zu dokumentieren, wiederverwendbare Komponenten bereitzustellen und Lernfortschritte nachvollziehbar zu machen.

## Über das Projekt
Dieses Repository fasst mehrere kleinere und mittlere Projekte zusammen, die unterschiedliche Technologien und Problemstellungen adressieren. Alle Projekte sind modular aufgebaut und gut dokumentiert, sodass sie wiederverwendbar sind.

## Features
- Sammlung von Unity-Projekten und Spielmechaniken  
- KI-gestützte NPC-Systeme (LLM-basiert, Entscheidungslogik, Interaktionen)  
- Erstellung und Anpassung von 3D-Modellen und Assets in Blender  
- Motion-Capture-Animationen (Nicken, Winken, Blinzeln)  
- Automatisierung von Build-Prozessen (Python, PowerShell)  
- Verarbeitung und Analyse von CAD/3D-Daten (Pixyz, Catia)  
- Dokumentation, Beispiel-Setups und Tutorials für jedes Projekt

- **Programmiersprachen:** C#, Python, PowerShell, Java

- **3D-Modellierung & Animation:** Blender, Motion Capture

## Schnellstart

### 1. Repository klonen
```bash
git clone https://github.com/hashemno/Quellcode.git
cd Quellcode
```
---

## 2. Unity WebGL-Projekte
- Öffne das gewünschte Unity-Projekt in **Unity 2022.3+**
- Gehe zu **File → Build Settings → WebGL → Build**
- Öffne `index.html` im Browser oder hoste die Build-Ordner-Inhalte über GitHub Pages

## 3. Python-/Automatisierungsskripte:
   - Stelle sicher, dass Python installiert ist
   - Abhängigkeiten ggf. mit `pip install -r requirements.txt` installieren
   - Skripte nach Bedarf ausführen

## Unterprojekte / Beispielprojekte

### AI-gesteuerte NPCs – mmmake GmbH & Schwarz Gruppe
- Unity-Programmierung von NPC-Interaktionen in C# und Python  
- Erstellung und Anpassung von 3D-Charakteren und Assets in Blender  
- Integration von Text-to-Speech und LLM-basierten Dialogsystemen  
- Umsetzung natürlicher Animationen (Nicken, Winken, Blinzeln)  

### Computeranimation / Blender-Projekte
- Modellierung komplexer Gebäude (z. B. Taj Mahal)  
- Charakteranimation mit Rigify  
- Erweiterung von Mocap-Szenen durch animierte Objekte und Simulationseffekte  
 
### UNO-Spiel – Objektorientierte Programmierung in Java
- Entwicklung eines UNO-Kartenspiels in Java  
- Nutzung von Klassen, Objekten, Interfaces und Vererbung  
- User Interface zur Interaktion mit dem Spieler  

### Pixyz Pipeline – CAD-Datenbereinigung
- Automatisierung von CAD-Datenaufbereitung in Pixyz  
- Import, Tesselierung, Reparatur, Materialzuweisung, Export  
- Erstellung wiederverwendbarer Python-Skripte
