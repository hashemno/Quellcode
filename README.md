Über das Projekt
----------------
Dieses Repository fasst mehrere kleinere und mittlere Projekte zusammen, die unterschiedliche Technologien und Probleme adressieren. Ziel ist es, Lösungen zu dokumentieren, wiederverwendbare Komponenten bereitzustellen und Lernfortschritte nachvollziehbar zu machen.

Features / Highlights
---------------------
- Sammlung von Unity‑Projekten / Spielmechaniken
- KI‑gestützte NPC‑Systeme (Verhalten, Entscheidungslogik)
- 3D‑Modelle & Assets
- Automatisierungsskripte für Build/Workflow‑Optimierung
- Tools zur Datenverarbeitung und Analyse
- Dokumentation und Beispiel‑Setups für jedes Projekt

Tech‑Stack (Kurz)
-----------------
- Hauptsprache: C#
- Game Engine: Unity (falls in Projekten verwendet)
- Weitere Tools: (z. B. Blender für 3D, Python/Node.js/PowerShell für Scripting) — bitte in jeweiligen Unterprojekten nachsehen
